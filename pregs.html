Qué es una red informática.

Se entiende como red a un conjunto de equipos conectado entre sí, permitiendo la intercomunicación entre ellos.
Esto permite enviar infromación, archivos, tareas, etc.

Qué es una red LAN.

Las siblas LAN se refieren a Local Area Network, lo que quiere decir que se trata de una red de corto alcance.
estas son las utilizadas a nivel particular.

Qué es una red WAN.

En este caso WAN hace referencia a Wide Area Network, que se trata de una red de redes, que abarca un área mayor.

Qué es Internet.

Se entiende internet como la red global descentralizada que interconecta a equipos de todo el mundo.

Qué es la Web.

Internet es una red global de computadoras interconectadas que se comunican entre si mediante un conjunto de protocolos estandar de comunicación. Es un sistema descentralizado que conecta millones de dispositivos en el mundo.

Qué es WWW

WWW es la abreviatura de World Wide Web, que es un sistema de informacion global en internet mediante el cual se puede acxceder a información, imagenes, videos y demás vinculados mediante enlaces.

Qué es el protocolo HTTP y HTTPS.

Http (Protocolo de Transferencia de HiperTexto) es el protocolo que se autiliza a nivel de usuarío para la tranferencia de datos entre páginas y usuarios.
Su variante https (s de seguro) sinifica que la comunicación está encriptada.

Página web estática vs página web dinámica.

"Una página web dinámica es un tipo de sitio web que genera contenido en tiempo real en función de la interacción del usuario.
A diferencia de las páginas web estáticas, que muestran el mismo contenido para todos los usuarios, las páginas web dinámicas pueden personalizarse y cambiar en tiempo real."

Dirección IP. Diferencia entre una dirección IP privada y pública.

La dirección IP es un numero que identifica el dispositivo en una red, con ella es posible conocer la ubicación del cliente o el tipo de dispositivo.
Hay de dos tipos, privada y publica, la de primer tipo identifica al dispositivo solo en la red local en la que se encuentra, mientras que la pública lo hace a nivel de internet.

¿Qué es el alojamiento web/hosting? Ejemplos.

Sabiendo que las páginas web se almacenan en servidores, el alojamiento o host se refiere al hecho de almacenar la información y mantenerla conectada a la red.
Normalmente de esto se encargan empresas dedicadas expresamente a esto, que poseen los servidores y los "alquilan".
